Polynomials Introduction

Polynomials are algebraic expressions with more than two terms.

The word poly comes from the Greek word polu meaning much or many. The word nomial means a name or term. Hence, polynomials = many terms!

Polynomial Terminology & Parts

Polynomials are multi-term algebraic expressions and consist of the following:

  • Constants - Non-variable values, like 21, -9, Pi, etc
  • Variables - Varying values, like x, y, z, a, b, etc
  • Exponents - Superscript numbers that can multiply a term by itself however many times, like x2 or 3y5

These parts can be combined to form the algebraic expression through any basic arithmetic operation except when a term is being divided by a variable or has a negative exponent.

The format of a term is as follows: axk   x reprsenting the variable, a being the coeffecient of the variable, and k being the exponent or degree of the term.

Quick reminder that a coeffecient is the integer in front of a variable. It multiples the variable by it's value. If there is no number however, the coeffecient value is 1, thus multiplying whatever variable there is by itself.

The Degree in poylnomials refers to the the largest exponent value present within the terms. Standard Form is the general layout in which terms are written in descending order from the LARGEST exponent value to the SMALLEST exponent value.

Stanard Form Example: 3x3 - 4x2 + 2 This polynomial is a cubic trionomial. This is because 3 is the largest exponent value and there are 3 terms.

Polynomial Degree Naming

Degree Name Description
Constant The exponent / degree value is 0. Constant Term.
Linear The exponent / degree value is 1. Linear term.
Quadratic The exponent / degree value is 2. term * term
Cubic The exponent / degree value is 3. term * term * term
Quartic The exponent / degree value is 4. term * term * term * term
Quintic The exponent / degree value is 5. term * term * term * term * term
6+ Any exponent / degree value that is 6 or above will simply be called the number then degree

Polynomial Term Naming

Name Description
Monomial A single term expression.
Binomial A expression containing two terms.
Trinomial An expression containing three terms.
Polynomial An expression containing four or more terms.

Examples of Polynomials

  • x2 + 8 - This example is a basic polynomial. It is a Quadratic Binomial because it has two terms and it's highest degree is 2.
  • 4x5 + 7y3 - 2x - 7 - This is a Quintic Polynomial. It contains 4 terms and the largest exponent value is 5.
