Trigonometric Functions - Overview

An easy way to remember the main 3 functions is using the mnemomic, SOHCAHTOA!
Sine = Opposite / Hypotenuse
Cosine = Adjacent / Hypotenuse
Tangent = Opposite / Adjacent

The functions are also considered the trigonometric ratios for right-triangles.

Function Abbreviation Relationship & Description
Sine sin Opposite / Hypotenuse - The sine value of a given angle is defined to be the opposite's side length over the hypotenuse's length.
Cosine cos Adjacent / Hypotenuse - The cosine value of a given angle is defined to be the adjacent's side length over the hypotenuse's length.
Tangent tan Opposite / Adjacent - The tangent value of a given angle is defined to be the opposite's side length over the adjacent's side length.
